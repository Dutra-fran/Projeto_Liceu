# Projeto_Liceu

Criador: Francisco Robson de O. Dutra
<br>Instagram: @robsondutra_

Esse projeto tem como principal função ajudar os professores da minha escola a serem mais produtivos. Nesse projeto, eu adicionei funções tais como:
<ul>
   <li>Cadastro e login dos alunos;</li>
   <li>Cadastro e login dos professores;</li>
   <li>Mecanismo de criação de prova, disponível para o professor;</li>
   <li>Mecanismo de criação do gabarito, disponível para o professor;</li>
   <li>Realização da prova, disponível para o aluno;</li>
   <li>O aluno, após realizar a prova, obterá instantaneamente sua nota;</li>
   <li>Mecanismo de feedback;</li>
   <li>O professor poderá visualizar os alunos que estão usando a plataforma;</li>
   <li>O professor poderá visualizar a nota que os alunos obtiveram em sua disciplina;</li>
   <li>Além de outras mecânicas.</li>
</ul>
