# Projeto_Liceu
Esse projeto tem como principal função ajudar os professores da minha escola a serem mais produtivos. Nesse projeto, eu adicionei funções tais como: cadastro e login dos alunos; cadastro e login dos professores; mecanismo de criação de prova, disponível para o professor; mecanismo de criação do gabarito, disponível para o professor; realização da prova, disponível para o aluno; o aluno, após realizar a prova, obterá instantaneamente sua nota; mecanismo de feedback; o professor poderá visualizar os alunos que estão usando a plataforma; o professor poderá visualizar a nota que os alunos obtiveram em sua disciplina; além de outras mecânicas.
